const user = {
  name: "Aman",
  age: 17
};

Object.freeze(user);

user.age = 18;       // ❌ change nahi hoga
user.city = "Delhi"; // ❌ add nahi hoga
delete user.name;    // ❌ delete nahi hoga

console.log(user);
// { name: "Aman", age: 17 }



Important baat ⚠️

Freeze shallow hota hai (sirf top level pe apply hota hai)

const user = {
  name: "Aman",
  address: {
    city: "Delhi"
  }
};

Object.freeze(user);

user.address.city = "Mumbai"; // ✅ change ho jayega!

👉 Isliye nested objects ko freeze karne ke liye deep freeze karna padta hai.



const user = {
  name: "Aman",
  address: {
    city: "Delhi"
  }
};

Object.freeze(user.address); // jis objects par lgega vo hi freez hofa bss


user.address.city = "Mumbai"; // not change ho jayega!
user.name ="rohit";
console.log(user) //{ name: 'rohit', address: { city: 'Delhi' } }

// seal meanthd
🔹 Object.seal() kya karta hai?

Object.seal() object ko partially lock karta hai 🔒

👉 Matlab:

❌ New properties add nahi kar sakte
❌ Properties delete nahi kar sakte
✅ Existing properties ko update kar sakte ho
🔹 Example
const user = {
  name: "Aman",
  age: 17
};

Object.seal(user);

user.age = 18;       // ✅ update ho jayega
user.city = "Delhi"; // ❌ add nahi hoga
delete user.name;    // ❌ delete nahi hoga

console.log(user);
// { name: "Aman", age: 18 }
🔹 Check kaise kare sealed hai ya nahi
console.log(Object.isSealed(user)); // true
🔹 freeze() vs seal() (simple difference)
Feature	freeze() ❄️	seal() 🔒
Add property	❌ नहीं	❌ नहीं
Delete property	❌ नहीं	❌ नहीं
Update property	❌ नहीं	✅ हाँ
💡 Easy samajh
freeze() = full lock (kuch bhi change nahi) 🔐
seal() = half lock (sirf update allowed) 🔓
